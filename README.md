# ST7789 RSDesign Color Dashboard

PironMan-style color dashboard for 172x320 ST7789 SPI displays.

Features:
- Multi-page rotating dashboard
- CPU / RAM rings
- Network graph
- Weather (OpenWeatherMap)
- Polished layout, no flicker

Hardware:
- Raspberry Pi
- ST7789 172x320 SPI display

Run:
```bash
python st7789_rsdesign_color_dashboard_allinone.py
